# Book Review Platform

This repository is bootstrapped as a monorepo for the Book Review Platform.

## Repository structure

- apps/: deployable applications
- packages/: shared libraries, UI components, and tooling
- docs/: project documentation and architecture notes
- scripts/: repository automation and setup helpers
- .github/: repository automation and Copilot instructions

## Current status

The repository has been initialized for future product development. Application code and feature implementations will be added in subsequent steps.
